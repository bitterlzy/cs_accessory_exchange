## 1. 修复内容

| 问题 | 文件 | 修复方式 |
|---|---|---|
| MySQL 从 WSL 连不上 localhost | .env | DATABASE_URL 改为 172.25.0.1（Windows 主机） |
| bcrypt + passlib 版本不兼容 | app/api/auth.py | 用直接 bcrypt.hashpw/checkpw 替代 CryptContext |
| Python 3.8 不支持 list[Model] | app/api/*.py | list[X] → List[X]，添加 from typing import List |
